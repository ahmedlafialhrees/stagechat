Fantasy City 3D Shooter
=========================
- مشروع ويب كامل (Three.js) — شخصية 3D تمسك سلاح، مدينة خيالية، وحوش، وموجات (مراحل).
- تحكم كمبيوتر: WASD للحركة + الماوس للتصويب/إطلاق. يتطلب Pointer Lock (اضغط ابدأ ثم انقر داخل الشاشة).
- تحكم جوال: عصا يسار للحركة، السحب يمين للتصويب، والضغط يمين للإطلاق.

تشغيل محلي:
-----------
افتح index.html مباشرة في المتصفح (Chrome/Safari/Firefox).

رفع على GitHub Pages:
---------------------
ارفع الملفات: index.html, style.css, main.js إلى المستودع.
من Settings → Pages → اختر Deploy from a branch (main /root).
